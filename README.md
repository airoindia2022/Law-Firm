# Integrity Legal Centre

A premium, modern, and responsive law firm website built with React, Vite, Tailwind CSS, and Framer Motion. This application showcases legal services, practice areas, team expertise, awards, and more with a focus on high-end aesthetics and user experience.

## ✨ Features

- **Responsive Design**: Fully optimized for desktops, tablets, and smartphones.
- **Dynamic Animations**: Smooth transitions and scroll-triggered animations using Framer Motion.
- **Service Showcases**: Detailed breakdown of various legal practice areas.
- **Team Profiles**: Insightful previews of the firm's legal expertise.
- **Interactive Contact Form**: Integrated with Web3Forms for seamless client communication.
- **Premium Aesthetics**: Curated color palettes, glassmorphism, and modern typography.
- **Multi-Location Support**: Highlighting hubs in Delhi and Lucknow.

## 🛠️ Tech Stack

- **Core**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [Material UI Icons](https://mui.com/material-ui/material-icons/)
- **UI Components**: [Material UI](https://mui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router DOM v6](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd law-firm
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production

Build the application for production:
```bash
npm run build
```
The optimized build will be generated in the `dist` folder.
